# Examples
This is a basic example script for testing multifidelity operations on wing data, which has 10 numerical columns and 4 different sources. The script goes through using the data_gen function as well as fitting the model to the wing data.

Problem: Wing - Multifidelity (1 HF 3 LF)

How to run: File should run without errors by pressing run file. Feel free to adjust the seed, data generation, and model inputs such as the combined kernel and its encoders to test different setups.

Expected output: Model metrics, training time.
